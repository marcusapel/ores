Commands List
=============

* WeCoResView : results visualization
* WeCoGui : main gui
* WeCoRes2Las : create las file from result
* WeCoRes2Csv : create csv file from result
* WeCoCheck : Check WeCo installation
* WeCoAddRegion : Add region to well file
* WeCoCM2Csv : Write Cost Matrix in a csv file
* WeCoWellList : Show Well List Content
* WeCoRegion2Data : create a log from region

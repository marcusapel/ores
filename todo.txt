OSDU ↔ Remote RDDMS: Pitfalls, TODOs & Admin Requests
Updated: 2026-05-10  (cleaned done items)
═══════════════════════════════════════════════════════════════════════

This doc captures lessons learned from the RDDMS ↔ OSDU catalog
roundtrip, plus improvements and admin items needed for production.

───────────────────────────────────────────────────────────────────────
1. PITFALLS & LESSONS LEARNED
───────────────────────────────────────────────────────────────────────

1.1  RDDMS REST API is EML-URI-centric
     ──────────────────────────────────
     REST listing returns {"uri": "eml:///dataspace('ds')/type(uuid)"}
     — no top-level UUID/ContentType keys.
     Fix: _parse_eml_entry() regex parser.

1.2  GET /resources wraps single object in a list
     Fix: Unwrap [obj] → obj in _rest_get_resource().

1.3  Underscore in dataspace name → manifest/build 500
     Server-side bug. Use names without underscore. See §3.2.

1.4  EPC XML ↔ RDDMS JSON schema divergence
     ETP --import-epc is the safest path. REST PUT needs hand-crafted
     JSON with $type annotations.

1.5  PropertyKind: two flavours
     StandardPropertyKind: {"Kind": "porosity"}
     LocalPropertyKind: {"LocalPropertyKind": {"Title": "..."}}
     Fix: _extract_property_kind() handles both.

1.6  Manifest builder omits WPC DatasetIDs
     POST /manifests/build produces WPCs without DatasetIDs.
     Fix: manifest_ingest.py auto-patches DatasetIDs after build.

1.7  Local RDDMS stack — architecture
     open-etp-client image (840 MB) = NestJS REST API, NOT a CLI.
     Includes manifest builder at POST /manifests/build.
     Env vars: RDMS_ETP_PROTOCOL, RDMS_ETP_HOST, RDMS_ETP_PORT,
               RDMS_REST_ROOT_PATH, RDMS_REST_PORT,
               RDMS_DATA_PARTITION_MODE, RDMS_JWT_SECRET.
     Local stack (docker-compose.yaml):
       postgres :5433  |  etp-server :9002  |  etp-client :3000
     Auth: any non-empty Bearer token accepted (authN=none).
     Manifest builder returns HTTP 201 (not 200) on success.

5. ORES ROUTE LIMITATIONS
───────────────────────────────────────────────────────────────────────

  /graphql — deep_search
     Array threshold filtering needs PG access for cell values.

  /graphql — federated_search
     Max 10 dataspaces auto-discovered; 5 hardcoded RESQML types.
     OSDU full-text search may produce false positives.

  /analyse — compare
     OSDU Search v2 can't filter nested array paths. Uses full-text
     workaround — may miss BDs.

  /keys — table view (STL resolution)
     Heuristic STL matching may pick wrong lookup table.

  /keys — structuremaps
     Time-domain Grid2d surfaces silently skipped.

  /strat — SMDA import/push
     Requires `az` CLI or SMDA_API_KEY. Non-functional without.

  /api/manifest/last
     In-memory only: max 50, 1h TTL, lost on pod restart.

  /search — wildcard kind queries
     ADME rejects kind:*. Auto-resolves but adds roundtrips.
